## ✍️ Run Locally
Clone the project 

```bash
  git clone https://github.com/viethung20101/DTU-Ticket.git
```

Go to the project directory

```bash
  cd DTU-TICKET
```

## 📦 Install
```sh
npm install
```

## 🚀 Usage
- Create `.env`
- Run server
```sh
yarn run dev
```
